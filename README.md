### **# My First Game 2020**



#### \## About The Project

This is simple video game made in **2020** with the help of a Brackeys tutorial.

In the game you play as a **sphere** where you have to **knock** three **cylinders** off the **stage** in three different levels.

* https://karamalhowari.itch.io/MyFirstGame
* https://github.com/KaramAlHowari/MyFirstGame



#### \## Controls

* **\[W,A,S,D]** to **roll** the sphere.
* **\[Spacebar]** to **jump**.
